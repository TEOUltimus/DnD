Dungeon Tile Mapper
Version 1.2.0
By Randal Meyer


You've probably seen our Dungeon Tiles previews in our monthly "Previews" article series and may even have scored a copy of these cool tiles. By now you probably have hundreds of cool dungeon designs running around your head. But how can you create these dungeons and share them with your friends?

Well a good friend of the D&D website, Randal "Jailoco" Meyer, has created a really cool drag-and-drop dungeon tile tool. And, he's given us a personalized copy for the D&D website as well as a nifty guide on its use. This tool includes all tiles from Dungeon Tiles I, Dungeon Tiles II: Arcane Corridors, Dungeon Tiles III: Hidden Crypts, Dungeon Tiles IV: Ruins of the Wild and will be updated with future sets.

(Launch Dungeon Tile Mapper!)

NOTE: Using our online version will let you save and load dungeons through the use of cookies, so if you would like to use that feature please have cookies turned on.

  • Download the Dungeon Tile Mapper version 1.2.0 now (4.17m ZIP). Unzip it into a directory on your machine and run the "default.htm" file in your web browser.

Dungeon Tile Mapper Guide

Generic

  • Caption (ToolBox): You can move the toolbox by dragging the caption.


  • Sub Title (Map, Settings, Tiles): This is the tab currently selected.


  • Tabs: These images change the currently viewed Tab in the ToolBox

Maps Tab

  • Map Selection DropDown: This selects a map to either load, export, or delete (using the appropriate icon).


  • Load Icon: This loads the selected Map.


  • Save Icon: This saves the current Map.


  • Delete Icon: This deletes the selected Map.


  • Export Icon: This creates the text needed to import the current Map on another computer.


  • Import Icon: This imports the given Map.


  • Print Icon: This opens up a window displaying only the Map for Printing.


Settings Tab

  • Grid Width DropDown: This selects the width of the Grid (12 to 72 in increments of 12).


  • Grid Height DropDown: This selects the height of the Grid (12 to 72 in increments of 12).


  • Set DropDown: This selects which TileSet you wish to set your count for.


  • Set Count DropDown: How many copies you have of the currently listed TileSet.


  • Maps Tab Icon Text Checkbox (checkbox): When checked, the Icons in the Maps Tab will have descriptive text.


  • Tiles Tab Icon Text Checkbox (checkbox): When checked, the Icons in the Tiles Tab will have descriptive text.


  • Anchor Delete Options (radio options): If Delete All is selected, all Anchored Tiles are deleted when their Anchor tile is deleted. If Detach All is selected, they are simply Detached before the Anchor is Deleted.


  • FreeForm Checkbox (checkbox): When checked, this removes the limit on the number of Tiles you can add from a Set.


Tiles Tab

  • Set DropDown: This selects which TileSet you wish to select from.


  • Side A DropDown: You use this to select the Tile you want. The text is the name of Side A.


  • Side B Text: This simply tells you the name of Side B of the selected Tile. Single sided Tiles do not have this.


  • Tiles used: This displays how many of the currently selected Tile you have used, and how many are available.


  • Side A Image: This is the Side A of the chosen Tile. Drag and Drop this image (or double click it) to add the Tile to the map.


  • Side B Image: This is the Side B of the chosen Tile. Drag and Drop this image (or double click it) to add the Tile to the map.


  • CounterClockwise Icon: This rotates the selected Tile counterclockwise.


  • Clockwise Icon: This rotates the selected Tile clockwise.


  • Delete Icon: This deletes the currently selected Tile from the map.


  • Send To Back Icon: This moves the selected Tile to the bottom layer.


  • Move Down Icon: This moves the selected Tile down one layer.


  • Move Up Icon: This moves the selected Tile up one layer.


  • Bring To Front Icon: This moves the selected Tile to the top layer.


  • Attach Icon: This attaches the currently selected Tile to the Tile below it.


  • Detach Icon: This detaches the currently selected Tile from its anchor.


Usage Notes

DoubleClick will flip the tile (hence no more flip button).

  • In FireFox (but not IE) you can Ctrl+F to flip it as well.

Rotate CounterClockwise, Delete, Rotate Clockwise are now icons.

  • Ctrl+{LeftArrow} and Ctrl+RightClick now rotate CounterClockwise while RightClick and Ctrl+{RightArrow} rotate Clockwise.
  • While anchored to another Tile, you cannot rotate a tile; it will rotate automatically if you rotate its anchor.

Layering Functions are now icons.

  • Ctrl+{UpArrow} now moves the Tile up one layer and Ctrl+{DownArrow} moves the Tile down one layer.

Saving and Deleting can only be done with cookies while online.

  • Save adds the name you give to the Map DropDown (overwriting any duplicates).
  • If using this offline, you will need to manually add any maps to the ExtensionsMaps.js file (there is already one in the file as an example).
  • When attaching a tile to another, it will attach to the Tile under its top left corner.
